## ğŸ“‹ Main Dashboard

### Tabs Available

#### 1. **Tasks Tab** (Default)
- View all tasks in List or Kanban view
- Filter by status, priority, assignee, date
- Search for specific tasks
- Bulk update/delete tasks
- Create new tasks

**List View**
- Table format with all task details
- Sort by any column
- Quick edit inline

**Kanban View**
- Drag-and-drop between columns
- Visual status tracking
- Quick task creation

#### 2. **Templates Tab**
- Create reusable task templates
- Set default priority, duration, reminders
- Organize with tags
- Quick task creation from templates

**How to Create Template**
1. Click "New Template"
2. Enter template name
3. Set type (Task, Call, Email, Meeting)
4. Set priority and duration
5. Add tags
6. Click "Create Template"

#### 3. **Workflows Tab**
- Automate task creation and updates
- Set triggers (deal stage change, contact import, etc.)
- Define custom actions
- Enable/disable workflows

**How to Create Workflow**
1. Click "New Workflow"
2. Enter workflow name
3. Select trigger event
4. Define action (JSON format)
5. Click "Create Workflow"

#### 4. **Settings Tab**
- Configure task system preferences
- Enable/disable notifications
- Enable/disable email reminders
- Enable/disable automation
